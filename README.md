# Costing

Aplicativo para gerir custos de projetos.

Features:
 - Criação de projeto
 - Criação de serviço em projeto
 - Calculo de budget
 - Categorias de projeto
 - Edição e exclusão de projeto
 - Edição e exclusão de serviços
